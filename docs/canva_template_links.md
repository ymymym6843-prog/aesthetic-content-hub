# IM AESTHETIC - Canva Template Search Links

아래 검색 링크에서 적합한 템플릿을 골라 IM AESTHETIC 브랜드 키트를 적용하세요.

## How to Use
1. 아래 링크 클릭 → 마음에 드는 템플릿 선택
2. "이 템플릿 사용" 클릭
3. 브랜드 키트에서 컬러/폰트 적용 (Brand Kit > IM AESTHETIC)
4. 텍스트를 IM AESTHETIC 콘텐츠로 교체
5. 사진을 실제 시술/고객 사진으로 교체

## Brand Kit Quick Reference

적용할 때 참고:
- **Primary**: #E8703A (기존 템플릿의 메인 컬러를 이것으로 교체)
- **Dark**: #1A1410 (텍스트, 다크 배경)
- **Cream**: #FFF8F0 (라이트 배경)
- **Font (EN)**: Playfair Display
- **Font (KR)**: Noto Sans KR (Pretendard 없으면)
- **Logo text**: IM AESTHETIC
- **Slogan**: 당신의 가장 빛나는 날을 준비합니다
