## Layout Types

- **Type A (상단 이미지 + 하단 텍스트)**: Main subject in upper 60-70% of frame, bottom 30-40% is clean/blurred/gradient for text overlay
- **Type B (다크 배경 + 중앙 여백)**: Dark background with large open center area for bold text overlay
- **Type C (전체 배경 텍스처)**: Entire image is subtle background texture, all areas open for text/infographic overlay
- **Type D (릴스 커버)**: Main subject centered, bottom 30% slightly darker/gradient for title text
