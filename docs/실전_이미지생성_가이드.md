# IM AESTHETIC - 실전 이미지 생성 가이드

> **실제 사진 우선 / AI = 보조 배경·일러스트 / Canva = 텍스트 + 레이아웃**
>
> ```
> [우선순위 1] 실제 사진 에셋 (인물·공간·시술 장면) — 리얼리티가 핵심
>     ↓ (실제 사진 없는 경우에만)
> [우선순위 2] AI 생성 이미지 (배경 텍스처, 일러스트, 무드 이미지)
>     ↓
> [Canva]  텍스트 (제목 1~2줄, 본문, CTA) + 로고 + 아이콘
> ```
>
> **이미지 소스별 적합 용도:**
> - **실제 사진 필수**: 전문가 얼굴, 공간, 시술 장면, B&A, 제품 실물, 고객 후기
> - **AI 생성 적합**: 인포그래픽 배경, 의료 일러스트(근막/피부 단면), 시즌 무드(벚꽃 등), 제품 분위기컷
> - **Canva 직접 제작**: 카드뉴스, 체크리스트, 프로세스 카드, 후킹 텍스트 슬라이드, CTA
>
> **보유 실제 사진 에셋** (`실제사진/` 폴더):
> - 인물: `단체사진.jpg`, `유수정.jpg`, `우연우.jpg`, `김은경.jpg`
> - 공간: `shop-1~3.jpeg`(리셉션/바), `shop-5.jpeg`(케어룸), `shop-6.jpeg`(라운지), `shop-7.jpeg`(파우더룸)

## 생성 전 체크

- [ ] 전문가 사진 준비: `유수정.jpg`, `우연우.jpg`, `김은경.jpg`, `단체사진.jpg` (실제사진/ 폴더)
- [ ] 공간 사진 준비: `shop-1~7.jpeg` (실제사진/ 폴더)
- [ ] 실제 사진으로 커버 가능한 슬라이드는 AI 생성 건너뛰기
- [ ] Start/Still 쌍은 **같은 채팅 세션**에서 연속 생성 (앵글 일치 필수)
- [ ] Still 생성 시 Start 이미지를 참조로 업로드하면 일관성 향상
- [ ] AI 생성 이미지 = 배경 전용 (텍스트/로고 포함 X)
- [ ] 텍스트/로고/CTA는 Canva에서 레이어로 추가

## Flow(나노바나나) 비율 설정 가이드

> Flow에서는 프롬프트 텍스트로 비율이 적용되지 않습니다.
> 반드시 **UI의 비율 선택 옵션**에서 설정하세요.

| 용도 | 최종 규격 | Flow 설정 | Canva 후처리 | 그리드 크롭 |
|------|-----------|-----------|-------------|------------|
| **피드 이미지** (화/목) | 4:5 (1080x1350) | **세로(9:16)** 으로 생성 | 1080x1350으로 크롭 | 좌우 32px 크롭 (3:4) |
| **릴스 프레임** (3W토~ Start/Still) | 9:16 (1080x1920) | **세로(9:16)** 그대로 사용 | 크롭 불필요 | (릴스탭 전용) |
| **릴스 커버** (3W토~) | 4:5 (1080x1350) | **세로(9:16)** 으로 생성 | 1080x1350으로 크롭 | 좌우 32px 크롭 (3:4) |

**크롭 팁:**
- 9:16으로 생성하면 세로가 넉넉 → 4:5로 잘라도 구도 유지
- Canva에서 `사이즈 조절` → 1080x1350 입력 → 원하는 영역 선택
- 피사체(인물/제품)가 **수직 중앙**에 오도록 구도 잡기
- **좌우 40px 이내에 중요 요소 배치 금지** (그리드 크롭 대비)

## 2026 인스타그램 그리드 세이프존 (3:4 직사각형 그리드)

> **2025년부터 인스타 프로필 그리드가 1:1(정사각형) → 3:4(세로 직사각형)로 변경되었습니다.**
> 4:5 피드 포스트는 세로가 거의 그대로 보이고, **좌우만 약 32px씩 크롭**됩니다.

### 그리드 크롭 시각화

```
[4:5 포스트 (1080x1350) → 3:4 그리드 (1015x1350)]

  32px                              32px
  잘림   ┌─── 그리드 보이는 영역 ───┐  잘림
  ←──→  │                         │  ←──→
 ┌──────│─────────────────────────│──────┐
 │      │                         │      │  ↑
 │      │      안전 영역            │      │  │
 │      │  (중요 요소는 여기에)      │      │  │
 │      │                         │      │  1350px
 │      │                         │      │  (세로 전부 보임!)
 │      │                         │      │  │
 │      │                         │      │  │
 │      │                         │      │  ↓
 └──────│─────────────────────────│──────┘
        └─────────────────────────┘
         ←──── 1015px 보임 ────→
```

### 콘텐츠별 그리드 노출 정리

| 콘텐츠 | 원본 비율 | 그리드 노출 | 크롭 영향 |
|--------|----------|-----------|---------|
| 피드 포스트 | 4:5 (1080x1350) | 3:4 (1015x1350) | **좌우 32px씩만 크롭, 세로 전부 보임** |
| 캐러셀 첫장 | 4:5 (1080x1350) | 3:4 (1015x1350) | 피드와 동일 |
| 릴스 커버 | 4:5 (1080x1350) | 3:4 (1015x1350) | 피드와 동일 |
| 릴스 (피드탭) | 9:16 → 4:5 크롭 | 3:4 (1015x1350) | 피드와 동일 |

### 디자인 규칙

1. **좌우 마진 40px 확보** — 그리드 크롭(32px) + 여유분
2. **피사체/핵심 요소는 수직 중앙 배치** — 세로는 안 잘리지만, 시각적 안정감 위해
3. **하단 텍스트 영역: 선택사항** — 글래스모피즘 또는 단색 웜화이트 반투명 중 선택. 기본은 실제 사진 위 간결한 타이포 1~2줄. 세로 크롭이 없으므로 하단 텍스트도 그리드에서 보임
4. **좌측 세로 악센트 라인: 좌측 여백 40px 이상 확보** — 왼쪽 크롭 대비
5. **로고(IM AESTHETIC) 우하단 배치 시: 우측 여백 40px 이상 확보**
