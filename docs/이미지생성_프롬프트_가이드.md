# IM AESTHETIC - 8주 이미지 생성 프롬프트 가이드

## 사용법

### Start/Still 쌍 생성 시 주의사항 (중요!)
- Start와 Still 프롬프트를 **같은 채팅 세션**에서 연속으로 생성하세요
- 먼저 Start를 생성한 후, **같은 대화에서** "Now create the COMPLETED version of this exact same scene" + Still 프롬프트를 입력
- 별도 세션에서 따로 생성하면 카메라 앵글/배경/조명이 달라질 수 있음
- 결과가 다르면, Still 생성 시 Start 이미지를 **참조 이미지로 함께 업로드**하고 "Match this exact camera angle and background" 추가

### Google AI Studio에서 직접 생성하기
1. https://aistudio.google.com 접속
2. 좌측 메뉴에서 **"이미지 생성"** 또는 **채팅**에서 이미지 생성 모델 선택
3. 아래 프롬프트를 복사하여 붙여넣기
4. **인물 프롬프트** (참조 사진 필요)인 경우: 전문가 사진을 함께 업로드
5. 생성된 이미지 다운로드

### ChatGPT DALL-E / Midjourney에서 생성하기
- 아래 프롬프트를 그대로 사용 가능
- Midjourney: 프롬프트 끝에 `--ar 4:5` (피드) 또는 `--ar 9:16` (릴스) 추가

## 톤앤매너 통일 규칙

### 비주얼 톤 (전 주차 공통)
- **조명**: 따뜻한 앰버/골든 톤 (차가운 블루 금지)
- **배경**: 웜 화이트, 베이지, 크림색 계열 통일
- **질감**: 리넨, 대리석, 원목 등 자연 소재
- **여백**: 충분한 여백, 미니멀 레이아웃
- **인물**: 따뜻하고 자연스러운 표정, 과한 포즈 금지

### 텍스트 톤
- **1~2주차**: 텍스트 최소화. 이미지 비주얼이 주인공. 텍스트는 브랜드명 + 간결한 1줄 카피만
- **3~5주차**: 정보성 텍스트 추가되지만 깔끔한 인포그래픽 스타일 유지
- **6~8주차**: 후킹 대형 텍스트 허용, 하지만 브랜드 컬러/폰트 규칙 유지

### Phase 구조
- **Phase 1 (1~2W)**: SNS 런칭 & 전문가 소개 — 릴스 없음, 이미지/캐러셀만
- **Phase 2 (3~5W)**: 이벤트·교육·후기 — 3W부터 릴스 시작
- **Phase 3 (6~8W)**: 전문성·프로모션 심화

### 피드 그리드 통일감 체크
```
1~3주차 피드 그리드 (9개 포스트):
┌──────────┬──────────┬──────────┐
│ 1W 화     │ 1W 목     │ 1W 토     │  리얼 스냅 +
│ 단체사진   │ 공간사진   │ 3인케어   │  웜톤 통일
│ (실제)    │ (실제)    │ (실제+AI) │
├──────────┼──────────┼──────────┤
│ 2W 화     │ 2W 목     │ 2W 토     │  인물 중심
│ 유수정     │ 우연우     │ 김은경     │  따뜻한 앰버톤
│ (실제)    │ (실제)    │ (실제+AI) │
├──────────┼──────────┼──────────┤
│ 3W 화     │ 3W 목     │ 3W 토     │  브랜딩 + 정보
│ 첫방문혜택  │ 첫방문과정  │ ASMR릴스  │  일관된 여백감
│ (실제+카드)│ (실제+AI) │ (촬영)    │
└──────────┴──────────┴──────────┘
→ 실제 사진 기반 + 톤앤매너(색감/여백/폰트) 통일로 브랜드 각인
→ AI 이미지는 카드뉴스 배경·일러스트에만 보조 활용
→ 1~2W는 릴스 없음 (캐러셀만), 3W부터 릴스 시작
```
