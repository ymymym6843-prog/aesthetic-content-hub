# IMGROUND 폰트 설정 가이드 (Canva 적용용)

> CSS 프리뷰는 약 1/3 스케일입니다. 아래는 **실제 Canva 캔버스에서 사용할 폰트 크기**입니다.

---

## 사용 폰트 3종

| 폰트 | 용도 | Canva에서 |
|------|------|----------|
| **Playfair Display** | 브랜드명 "IMGROUND", EP 넘버 | 검색: "Playfair Display" (기본 탑재) |
| **Pretendard** | 제목, 본문, 뱃지, 불릿 | Canva에 없으면 → **Noto Sans KR** 대체 |
| **Noto Serif KR** | 감성 훅 제목 (일정표 등) | 검색: "Noto Serif KR" (기본 탑재) |

## Canva 팁

- **Pretendard**가 Canva에 없으면 → **Noto Sans KR** 사용 (가장 유사)
- 자간(letter-spacing)은 Canva에서 **"간격"** 설정으로 조절
- 음수 자간(-1.5px 등)은 Canva에서 간격을 **마이너스**로 설정
- 브랜드명 "IMGROUND"는 항상 **대문자**로 표기
