# IM AESTHETIC - 릴스 영상 제작 파이프라인

> 3W~8W 토요일 릴스 총 6편을 위한 실전 제작 가이드
> (1W, 2W는 릴스 없음 — 캐러셀/이미지 콘텐츠만 운영)
> **실제 영상 촬영 우선** → 촬영 불가 시 AI 이미지 기반 슬라이드쇼 릴스로 대체

### 릴스 제작 방식 선택

| 방식 | 적합한 경우 | 퀄리티 |
|------|-----------|--------|
| **A. 실제 촬영** (권장) | 스마트폰으로 촬영 가능한 환경 | ★★★★★ |
| **B. AI 이미지 → 모션** | 실제 촬영 불가, 무드/분위기 영상 | ★★★☆☆ |
| **C. 사진 슬라이드쇼** | 촬영·AI 모두 어려울 때 | ★★☆☆☆ |

**A방식 (실제 촬영):** 스마트폰 세로(9:16) + 삼각대 고정 + 자연광/조명. ASMR은 외장 마이크 권장.
**B방식 (AI→모션):** 아래 파이프라인 그대로 진행.
**C방식 (슬라이드쇼):** 실제 사진(shop-1~7.jpeg, 전문가 사진) + AI 무드 이미지 조합 → CapCut에서 줌/패닝 효과 + BGM.

## 전체 플로우

```
[1단계] 스토리보드 확인
  실전_이미지생성_가이드.md에서 해당 주차 Start/Still + 촬영가이드 확인
    ↓
[2단계] AI 이미지 생성
  Gemini / Midjourney로 Start·Still 쌍 생성
    ↓
[3단계] 이미지 후보정
  Photoshop or Canva로 색감 통일 + AI티 제거
    ↓
[4단계] 모션 영상 변환
  Kling AI로 정지 이미지 → 움직이는 영상 변환
    ↓
[5단계] BGM 생성
  Suno AI로 브랜드 톤 배경음악 생성
    ↓
[6단계] 최종 편집
  CapCut으로 클립 조합 + 텍스트 + 효과음 + 엔딩
    ↓
[7단계] 내보내기 + 업로드
  MP4 (1080x1920, 30fps) → 인스타그램 릴스
```

## 1단계: 스토리보드 확인

각 주차 릴스의 씬 구조는 이미 설계되어 있습니다.
**릴스는 3W부터 시작합니다. 1W, 2W는 릴스가 없습니다.**

### Phase 구분

| Phase | 주차 | 릴스 여부 | 전략 |
|-------|------|-----------|------|
| **Phase 1** (도입) | 1W, 2W | 릴스 없음 | 이미지/캐러셀로 브랜딩 집중 |
| **Phase 2** (확장) | 3W, 4W, 5W | 릴스 시작 (3편) | ASMR·언박싱·홈케어로 참여 유도 |
| **Phase 3** (전환) | 6W, 7W, 8W | 릴스 계속 (3편) | 후킹 강화·CTA 본격화 |

### 주차별 릴스 목록 (총 6편)

| 주차 | 릴스 주제 | 핵심 전환 |
|------|-----------|-----------|
| 1W | **릴스 없음** | — |
| 2W | **릴스 없음** | — |
| 3W | 고객 맞이 준비 ASMR — 오직 한 분을 위한 방 | 빈 베드 → 완벽 세팅 완료 |
| 4W | 프리미엄 앰플 언박싱 & 제형 시연 | 밀봉 상자 → 세팅 완료 |
| 5W | 집에서 하는 1분 괄사 마사지 (홈케어 팁) | 괄사 도구 준비 → 마사지 완료 |
| 6W | 샵 마감 루틴 | 사용 후 트레이 → 깔끔 정리 완료 |
| 7W | 스파 물소리 ASMR | 빈 족욕기 → 허브 우러난 완성 |
| 8W | 끊임없는 연구, 3인의 스터디 로그 | 빈 책상 → 공부 완료 |

**확인할 것:**
- `실전_이미지생성_가이드.md`에서 해당 주차 `_start.png`, `_still.png` 프롬프트
- 각 프롬프트 아래 `촬영/편집 가이드` 블록의 전환 액션 5단계

## 2단계: AI 이미지 생성

### 도구 선택

| 도구 | 장점 | 단점 |
|------|------|------|
| **Gemini (나노바나나)** | 무료, 프롬프트 한국어 지원 | 퀄리티 변동 |
| **Midjourney v7** | 최고 퀄리티, 일관성 우수 | 유료 ($10/월~) |

### 생성 규칙

```
필수 규칙:
1. Start와 Still은 반드시 같은 세션(채팅)에서 연속 생성
2. Still 생성 시 "SAME camera angle, SAME lighting" 반복 명시
3. Still 프롬프트에 Start 이미지를 참조로 업로드하면 일관성 향상
4. 비율: 9:16 (1080x1920) 고정
5. 한 번에 마음에 드는 결과가 안 나오면 3회까지만 재생성
```

### Midjourney 사용 시 추가 팁

```
프롬프트 끝에 추가:
--ar 9:16 --style raw --s 250 --v 7

일관성 유지:
--cref [Start 이미지 URL]  ← Still 생성 시 Start를 참조로 걸기
--cw 100                    ← 참조 강도 최대
```

## 3단계: 이미지 후보정

> "AI 느낌 제거가 아니라 브랜드 색감 고정 작업"

### Photoshop (권장)

```
체크리스트:
[ ] 색온도 통일: 따뜻한 앰버톤 (화이트밸런스 +10~15 웜)
[ ] 피부 질감 정리: AI 특유의 왁스 느낌 제거 (텍스처 레이어 추가)
[ ] 빛 반사 정교화: 하이라이트 자연스럽게 (Dodge & Burn)
[ ] 브랜드 컬러 확인: #FFF8F0(배경), #FF8C42(포인트), #333(텍스트)
[ ] 로고 위치: 우하단 "IM AESTHETIC" 오렌지
[ ] Start↔Still 색감 완벽 일치 확인
```

### Canva (간편 대안)

```
Canva에서 가능한 보정:
- 밝기/대비/채도 조절
- 색온도 슬라이더 → 웜톤 +15
- 필터: 없음 (일관성 위해 필터 사용 금지)
- "배경 제거" 기능으로 불필요한 요소 정리
```

**핵심: Start와 Still의 색감이 달라지면 영상 전환 시 부자연스러움 → 반드시 같은 보정값 적용**

```

길이: 5초 (무료) / 10초 (프로)
```

## 5단계: BGM 생성 (Suno AI)

> "영상은 시각보다 청각이 분위기를 결정합니다"

### Suno 기본 설정

```
접속: https://suno.com
모드: Custom (직접 스타일 지정)
길이: 30초 (릴스 맞춤)
```

### IM AESTHETIC 브랜드 BGM 프롬프트

**ASMR 릴스용 (3W, 7W):**
```
Style: ambient, minimal, spa
BPM: 60-70
Instruments: soft piano, gentle pad synth, subtle wind chimes
Mood: warm, calming, premium
No vocals, no drums, no sudden changes
30 seconds, fade out ending
```

**케어/시술 릴스용 (4W, 5W, 6W):**
```
Style: lo-fi ambient, organic
BPM: 70-80
Instruments: acoustic guitar fingerpicking, soft Rhodes piano, nature sounds
Mood: trustworthy, professional, gentle
No vocals, minimal percussion
30 seconds, fade out ending
```

**스터디/성장 릴스용 (8W):**
```
Style: lo-fi study beats, warm
BPM: 75-85
Instruments: mellow piano, soft vinyl crackle, gentle bass
Mood: focused, cozy, aspirational
No vocals
30 seconds, fade out ending
```

### BGM 선택 기준

```
체크리스트:
[ ] BPM 80 이하인가? (고급 브랜드는 느린 템포)
[ ] 갑작스러운 전환이 없는가?
[ ] 과한 감정 표현이 없는가? (드라마틱 금지)
[ ] Start→Still 전환 타이밍에 자연스러운가?
[ ] 페이드아웃이 깔끔한가?
```

### 대안: 저작권 프리 BGM

Suno 대신 사용 가능한 무료 소스:
- **인스타그램 오디오 라이브러리** — 릴스 편집 시 직접 추가 (가장 안전)
- **Artlist 무료 플랜** — 상업용 가능
- **YouTube 오디오 라이브러리** — CC0 트랙

## 6단계: 최종 편집 (CapCut)

### CapCut 프로젝트 설정

```
새 프로젝트:
- 비율: 9:16
- 해상도: 1080x1920
- 프레임레이트: 30fps
- 배경: 검정
```

### 릴스 타임라인 구조 (15~30초)

```
[0.0s ~ 0.5s]  브랜드 인트로 (선택)
                IM AESTHETIC 로고 페이드인
                ↓
[0.5s ~ 1.5s]  첫 1초 후킹
                Start 이미지/영상 등장
                텍스트: 릴스 제목 (예: "오늘의 세팅 시작")
                ↓
[1.5s ~ 8.0s]  Start 모션 클립
                Kling에서 생성한 Start 영상
                ASMR 효과음 레이어 추가
                ↓
[8.0s ~ 9.0s]  전환 효과
                디졸브 or 모핑 (0.5~1초)
                ↓
[9.0s ~ 16.0s] Still 모션 클립
                Kling에서 생성한 Still 영상
                "완성" 느낌의 텍스트 오버레이
                ↓
[16.0s ~ 18.0s] 엔딩 CTA
                 "IM AESTHETIC"
                 "DM으로 상담 문의"
                 로고 + 페이드아웃
```

### 주차별 전략에 따른 편집 차이

**Phase 1 (1W~2W): 릴스 없음**
```
- 이 기간에는 릴스를 제작하지 않습니다.
- 이미지/캐러셀 콘텐츠로 브랜드 인지도 구축에 집중합니다.
```

**Phase 2 — 3W~5W (브랜딩 + 참여 도입):**
```
- 텍스트: 최소화 (브랜드명 + 한 줄만)
- 전환: 부드러운 디졸브
- 효과: 없음~가벼운 자막 애니메이션
- 엔딩: 로고 + 가벼운 CTA ("이런 관리 받아보셨나요?")
```

**Phase 3 — 6W~8W (후킹 본격화):**
```
- 텍스트: 첫 1초 대형 후킹 텍스트 ("이거 실화?")
- 전환: 빠른 컷 or 모핑
- 효과: 강조 텍스트 팝업, 사운드 싱크
- 엔딩: CTA 강화 ("지금 DM 주세요")
```

### CapCut 텍스트 설정

```
메인 텍스트:
- 폰트: Pretendard Bold (또는 CapCut 내 "고딕 Bold")
- 크기: 화면 너비의 70~80%
- 색상: #FFFFFF (흰색) + 그림자
- 위치: 화면 중앙~상단 1/3
- 애니메이션: 페이드인 (0.3초)

서브 텍스트:
- 폰트: Pretendard Regular
- 크기: 메인의 50%
- 색상: #FFF8F0 (따뜻한 화이트)
- 위치: 메인 텍스트 바로 아래

IM AESTHETIC 로고:
- 위치: 우하단 고정
- 크기: 화면 너비의 15%
- 투명도: 70%
- 전체 영상에 걸쳐 항상 표시
```

### 효과음 레이어 (ASMR 릴스)

```
추천 효과음 (CapCut 내장 or 별도 녹음):
- 린넨 바스락: fabric rustling
- 크림 디스펜서: pump click + squelch
- 물소리: gentle water splash
- 오일 한 방울: single drop
- 페이지 넘김: paper flip

효과음 볼륨: BGM의 150~200% (ASMR에서는 효과음이 주인공)
BGM 볼륨: 전체의 20~30% (배경으로만)
```

## 주간 릴스 제작 타임라인

> **참고: 1W, 2W에는 릴스 제작이 없으므로 이 파이프라인은 3W부터 적용합니다.**

```
BGM 선택/생성:                   10분 (첫 주에 만들어두면 재사용)

────────────────────────────────
총:                              약 1시간 40분/편
```

### 효율화 팁

```
1. BGM은 3종만 만들어두고 6주간(3W~8W) 로테이션
   - ASMR용 / 케어용 / 스터디용

2. CapCut 템플릿 저장
   - 3W(첫 릴스)에 만든 편집 구조를 "템플릿으로 저장"
   - 4W부터는 클립만 교체 → 시간 절반으로 단축

3. Kling 일괄 생성
   - 토요일 아침에 Start 6개 + Still 6개 한꺼번에 돌리기
   - 생성 대기 중 다른 작업 진행

4. 브랜드 색감 프리셋 저장
   - Photoshop: 액션(Action)으로 저장
   - Canva: "내 스타일" 필터로 저장
   - 매번 수동 보정 불필요
```
