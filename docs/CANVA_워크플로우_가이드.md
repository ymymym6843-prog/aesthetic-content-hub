# IM AESTHETIC - Canva 후작업 워크플로우 가이드

## 전체 프로세스

```
[Step 1] generate_all.py 실행 → Gemini AI 베이스 이미지 생성
    ↓
[Step 2] overlay_text.py 실행 → 기본 한국어 텍스트 + 로고 오버레이 (선택)
    ↓
[Step 3] Canva 후작업 → 캐러셀 레이아웃, 타이포그래피, 후킹 슬라이드 완성
    ↓
[Step 4] 최종 검수 → 인스타그램 업로드
```

## Step 1: AI 이미지 생성 (자동)

```bash

### 이미지 소스 우선순위 (중요!)

```
[1순위] 실제 사진 — 전문가 인물, 공간, 시술, B&A, 제품 실물, 고객 후기
[2순위] AI 생성 — 배경 텍스처, 의료 일러스트, 시즌 무드, 제품 분위기컷
[3순위] Canva 직접 제작 — 카드뉴스, 인포그래픽, 체크리스트, 후킹 텍스트
```

### 보유 실제 사진 에셋 (`실제사진/` 폴더)
- 인물: `단체사진.jpg`, `유수정.jpg`, `우연우.jpg`, `김은경.jpg`
- 공간: `shop-1~3.jpeg`(리셉션/바), `shop-5.jpeg`(케어룸), `shop-6.jpeg`(라운지), `shop-7.jpeg`(파우더룸)

### AI 생성이 적합한 것
- 인포그래픽 배경 텍스처 (웜톤, 추상)
- 의료 일러스트 (근막 연결도, 피부 단면도)
- 시즌 무드 이미지 (벚꽃, 봄 분위기)
- 제품 분위기컷 (앰플, 오일 등 — 실물 촬영이 없을 때만)
- 릴스 Start/Still 연속성 이미지 (실제 영상 촬영 불가 시)

### AI 생성 비적합 → 실제 사진 필수
- 전문가 얼굴 (이미 보유 에셋 활용)
- 실제 공간 (이미 보유 에셋 활용)
- 고객 B&A, 후기 (리얼리티가 핵심)
- 제품 실물 (브랜드 신뢰 문제)

### Canva 필수 작업
- 한국어 타이포그래피 (AI는 한국어 텍스트 정확도 낮음)
- 캐러셀 슬라이드 간 톤 통일
- 후킹 첫 슬라이드의 대형 텍스트 디자인
- CTA 마지막 슬라이드
- 실제 사진 위 간결한 타이포 오버레이 (1~2줄)

## Step 3: Canva 후작업 상세 가이드

### A. Canva 프로젝트 세팅

#### 브랜드 킷 등록
| 항목 | 설정값 |
|------|--------|
| 메인 컬러 | Warm White `#FFF8F0` (70%) |
| 보조 컬러 | Charcoal `#333333`, Beige `#F5E6D3` (25%) |
| 포인트 컬러 | Soft Orange `#FF8C42` (5%) |
| 메인 폰트 | Pretendard Bold (제목) |
| 본문 폰트 | Pretendard Regular (본문) |
| 로고 | "IM AESTHETIC" 텍스트 로고 (오렌지) |

#### 템플릿 사이즈
- **피드 캐러셀**: 1080 x 1350px (4:5)
- **릴스 커버**: 1080 x 1350px (4:5)
- **릴스 프레임**: 1080 x 1920px (9:16)

#### 2026 그리드 세이프존 (필독)
> 프로필 그리드가 **1:1 → 3:4**로 변경됨 (2025~)
> 4:5 포스트는 좌우 32px씩만 크롭, **세로는 전부 보임**

- **좌우 마진 40px 이상 확보** (그리드 크롭 32px + 여유분)
- 로고/텍스트가 좌우 끝에 닿지 않도록 주의
- 세로 악센트 라인은 좌측 40px 이후부터 시작
- 하단 요소(글래스모피즘 바, CTA)는 세로 크롭 없으므로 안전

### B. 캐러셀 디자인 포맷별 작업법

#### 단계별 디자인 포맷 배치
| 단계 | 주차 | 사용 포맷 | 이유 |
|------|------|----------|------|
| Phase 1: 브랜딩 집중 (릴스 없음) | 1~2W | 매거진 에디토리얼형 **통일** | 피드 그리드 톤 통일, 프리미엄 인상 |
| Phase 2: 참여 도입 + 릴스 시작 | 3~5W | 매거진형 + 카드 인포그래픽형 혼용 | 릴스 도입, 정보 콘텐츠 확대 |
| Phase 3: 후킹 본격 | 6~8W | 매거진형 + 파노라마형 + 플립형 다양화 | 스와이프 유도, 차별화 |

#### 1) 매거진 에디토리얼형 (1~2W 메인 포맷)
**적용 대상**: 1~2W 전체 캐러셀, 6W 화(차트 컨퍼런스), 7W 화(웨딩 케어)

```
Canva 작업 순서:
1. AI 생성 이미지를 배경으로 배치 (투명도 90~100%)
2. 상단 1/3에 얇은 산세리프 제목 배치 (좌우 40px 마진 확보)
3. 하단에 2~3줄 서브 카피
4. 좌측 40px 지점에 세로 오렌지 악센트 라인 (3px) — 그리드 크롭 밖으로 배치
5. 우하단 "IM AESTHETIC" 로고 (우측 40px+ 여백 확보)
```

**레이아웃 참고** (좌우 40px 세이프존 확보):
```
┌──┬─────────────────┬──┐
│  │ [AI 배경 이미지]  │  │
│  │                  │  │
│  │ 유수정 전문가     │  │ ← Pretendard Bold 42pt
│  │ 메디컬 스킨케어   │  │ ← Pretendard Regular 24pt
│  │                  │  │
│  │ "피부가 스스로    │  │ ← 이탤릭/인용 스타일
│  │  회복하는 힘"     │  │
│  │                  │  │
│  │    IM AESTHETIC  │  │ ← 오렌지 12pt
└──┴─────────────────┴──┘
 ↑ 좌우 40px 마진       ↑
   (그리드 크롭 대비)
```

#### 2) 시임리스 파노라마형 (6W 이후 도입)
**적용 대상**: 6W 화(차트 컨퍼런스) — 1~2W에서는 사용하지 않음 (그리드 통일감 우선)

```
Canva 작업 순서:
1. 캔버스 크기: 3240 x 1350px (3슬라이드 연결) 또는 2160 x 1350px (2슬라이드)
2. AI 이미지를 배경으로 깔기 (전체 파노라마)
3. 텍스트/아이콘을 슬라이드 경계를 넘어가도록 배치
4. 완성 후 1080px 단위로 잘라서 개별 슬라이드로 내보내기
```

#### 3) 카드 인포그래픽형 (3W부터 도입)
**적용 대상**: 3W 목(첫 방문 프로세스), 4W 목(위생 철칙), 5W 화(B&A), 5W 목(자가진단), 6W 목(성분 비교), 8W 목(4월 캘린더)

```
Canva 작업 순서:
1. 깨끗한 흰색/크림색 배경 설정
2. AI 생성 인포그래픽을 참고하되, Canva 요소로 재구성
3. 아이콘: Canva 기본 아이콘 세트 활용 (라인 스타일)
4. 체크박스, 그래프 등은 Canva 차트 요소 사용
5. 컬러: 오렌지 포인트만 사용, 나머지 차콜+화이트
```

#### 4) 질문-답변 플립형 (6W 이후 도입)
**적용 대상**: 7W 목(미신 타파) — 후킹 본격화 단계에서 사용

```
Canva 작업 순서:
Slide 1 (질문):
  - 큰 볼드 텍스트: "아파야 효과 있다?"
  - 배경: 다크 차콜 (#333)
  - 큰 "?" 아이콘

Slide 2 (답변):
  - "NO!" 큰 텍스트 + 팩트 설명
  - 배경: 밝은 화이트 (#FFF8F0)
  - 오렌지 체크마크

  ※ 극적인 대비로 스와이프 유도
```

### C. 후킹 슬라이드 디자인 규칙 (단계별)

#### Phase 1 (1~2W): 후킹 슬라이드 없음 — 브랜딩 집중, 릴스 없음
- 첫 슬라이드 = 브랜드 비주얼 자체가 임팩트
- 매거진 에디토리얼형으로 톤 통일
- 피드 그리드 6~9개가 동일한 따뜻한 톤으로 채워지는 것이 목표
- 릴스 없이 이미지/캐러셀만으로 브랜드 인상 확립

#### Phase 2 (3~5W): 가벼운 후킹 — 참여 도입, 릴스 시작
- 3W 토에서 첫 릴스(ASMR) 도입
- 별도 후킹 슬라이드 없이, 본문 첫 줄이 자연스럽게 후킹 역할
- 체크리스트/데이터 등 콘텐츠 자체가 참여 유도

#### Phase 3 (6~8W): 후킹 본격화
```
원칙:
- 텍스트 8~10자 이내 (한눈에 읽히는 길이)
- 폰트 크기: 60~80pt (화면의 40% 이상 차지)
- 배경: AI 생성 이미지 위에 반투명 오버레이
- 호기심 유발형 (질문 or 의외성)
- 클릭베이트 X, 소프트 큐리오시티 O
```

### E. 릴스 후작업 (CapCut / Canva Video)

> **참고**: 릴스는 3W부터 시작됩니다. 1~2W에는 릴스가 없습니다.

릴스는 Start + Still 이미지를 영상으로 편집:

```
편집 순서:
1. Start 이미지 → 2~3초 정지
2. 부드러운 전환 효과 (디졸브 or 모핑) 0.5초
3. Still 이미지 → 3~4초 정지
4. (선택) 중간에 실제 촬영 영상 삽입
5. 엔딩: 로고 + CTA 카드 2초

오디오:
