```

warm beige and dusty gold tones, soft pastel accents,
centered composition, clean cream background (#FFF8F0),
elegant premium spa aesthetic, Korean beauty editorial style,
muted palette, subtle gold highlights, hand-drawn feel,
no text, no border, transparent background optional
```

## 💡 사용 팁

### Canva Magic Media 사용법
1. Canva 에디터 → 왼쪽 메뉴 "앱" → "Magic Media" 검색
2. "이미지" 탭 선택
3. 프롬프트 복붙 → 스타일: "수채화" or "일러스트레이션"
4. 비율: **1:1 (정사각)** 권장
5. 생성된 이미지를 템플릿에 드래그

### Midjourney 사용법
```
/imagine [프롬프트] --ar 1:1 --style raw --v 6
```
- `--ar 1:1` : 정사각형 비율
- `--style raw` : 자연스러운 일러스트
- `--v 6` : 최신 버전

### DALL·E 3 (ChatGPT Plus)
```
이 프롬프트로 이미지를 만들어줘: [프롬프트 복붙]
1:1 정사각 비율, 투명 배경 옵션으로.
```

### 일관성 유지 팁
- **16장 모두 같은 AI 도구 사용** (스타일 일관성)
- **Midjourney**: `--seed [숫자]`로 같은 시드 반복 사용하면 톤 일치
- **DALL·E**: 첫 장 만든 후 "같은 스타일로" 지시어 포함
- **Canva**: "브랜드 스타일" 설정에 브랜드 컬러 등록 후 생성

### 사후 보정 (필요 시)
- **배경 제거**: Canva의 "배경 제거" 기능 (Pro)
- **색상 통일**: Canva "색상 조정" → 크림 계열 필터
- **크롭**: 중앙 정사각 크롭
