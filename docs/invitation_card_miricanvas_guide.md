# 미리캔버스 초대장 제작 가이드

`invitation_card.html` 기반 — 엽서형 초대장 (앞/뒷면)

---

## 캔버스 설정

- **크기**: 102mm × 152mm (세로형 엽서)
- **DPI**: 300dpi (인쇄용)
- **픽셀 기준**: 약 1205 × 1795 px
- **여백(블리드)**: 상하좌우 3mm 재단 여유 권장
- **파일**: 앞면 1p, 뒷면 1p (총 2페이지)

---

## 브랜드 컬러 팔레트 (6색)

| 용도 | HEX | 비중 |
|------|-----|------|
| Brand Orange (CTA/포인트) | `#E8703A` | 15~25% |
| Cream (앞면 배경) | `#FFF8F0` | 기본 |
| Back Cream (뒷면 배경) | `#FEFCFA` | 기본 |
| Warm Beige (구분선) | `#F5EDE0` / `#E8DDD0` | 서브 |
| Gold Accent (장식선/밑줄) | `#D4A04A` / `#C4A882` | 3~5% |
| Body Text (제목/본문) | `#3D2E24` | 메인 |
| Sub Text (캡션) | `#9B8A7A` | 보조 |

> **주의**: Orange(#E8703A)와 Gold(#D4A04A)는 인접 배치 금지 (톤 충돌)

---

## 폰트 설정

미리캔버스 기본 폰트 중 아래 매핑으로 설정:

| 용도 | 폰트 | Weight | 비고 |
|------|------|--------|------|
| 훅 제목 ("초대합니다") | **본명조 / Noto Serif KR** | Light (300) | 감성 유입 |
| 브랜드마크 ("IM AESTHETIC") | **Playfair Display** | Regular (400) | 영문 한정 |
| 콘텐츠 제목 ("초대장 이용 안내") | **Pretendard** | SemiBold (600) | 정보 전달 |
| 본문 (greeting, step 문구) | **Pretendard** | Regular (400) | 가독성 |
| 강조/라벨 | **Pretendard** | SemiBold (600) | |
| 숫자/배지 | **Pretendard** | Bold (700) | |

> 미리캔버스에 `Pretendard` 없으면 **본고딕(Noto Sans KR)** 으로 대체 가능

## 앞면 (Front) 레이아웃

### 배경
- 색상: `#FFF8F0`
- 중앙 상단 은은한 방사형 그라디언트 (Orange 4% 투명도) — 선택사항

### 요소 배치 (위에서 아래)

1. **모서리 장식선** (좌상단 + 우하단)
   - 크기: 18mm × 18mm
   - 색상: `#D4A04A`, 투명도 50%
   - 선 두께: 0.8pt
   - L자 형태 모서리선

2. **장미 일러스트** (우상단 + 좌하단)
   - 크기: 22mm × 22mm
   - 투명도: 70%
   - 좌하단은 180도 회전
   - 미리캔버스 "장미" 검색 또는 라인 아트 스타일 사용

3. **IM AESTHETIC 로고**
   - 폭: 28mm
   - 중앙 정렬
   - 기존 로고 파일 업로드 (`/logo-*.svg` 사용)

4. **이름 필드** (로고 하단 5mm)
   ```
   ___________   님
   ```
   - 밑줄: 길이 32mm, 두께 0.6pt, 색상 `#D4A04A`
   - "님": 본명조 Light, 13pt, `#3D2E24`
   - 자간 0.05em

5. **골드 구분선**
   - 길이 18mm, 두께 0.6pt, `#D4A04A`

6. **"초대합니다"**
   - 본명조 Light, **24pt**, `#3D2E24`
   - 자간 **0.18em** (넓게)

7. **"소중한 분께 드리는 특별한 초대"**
   - Pretendard Medium, 11.5pt, `#E8703A`
   - 자간 0.06em

8. **짧은 구분선**
   - 길이 6mm, 두께 0.5pt, `#D4A04A`

9. **인사말 (greeting)**
   - Pretendard Regular, 11pt, `#3D2E24`
   - 줄 간격 1.95 (약 21pt)
   - 중앙 정렬, 최대 폭 70mm
   ```
   바쁜 일상 속 나를 위한 시간,
   IM에스테틱이 특별한 케어를 준비했습니다.

   아름다운 변화를 느껴보세요.
   ```

10. **하단 주소/전화번호** (하단 8mm 위치)
    - 주소: Pretendard Regular, 8pt, `#9B8A7A`
      → `대구 수성구 범어동 마크팰리스 2층`
    - 전화번호: Pretendard SemiBold, 8.5pt, `#3D2E24`
      → `053-241-3855`

## 뒷면 (Back) 레이아웃

### 배경
- 색상: `#FEFCFA`
- 여백: 상/좌/우 9mm, 하단 7mm

### 요소 배치 (위에서 아래)

1. **헤더 영역**
   - "IM AESTHETIC": Playfair Display, 8pt, `#C4A882`, 자간 0.25em, 대문자
   - "초대장 이용 안내": Pretendard SemiBold, 13pt, `#3D2E24`, 자간 0.06em
   - 하단 구분선: 0.4pt, `#E8DDD0`
   - 중앙 정렬

2. **이용 안내 4단계** (번호 뱃지 + 텍스트)
   - 뱃지: 원형 Ø5mm, 배경 `#E8703A`, 숫자 흰색 Pretendard Bold 8.5pt
   - 텍스트: Pretendard Regular, 10pt, `#3D2E24`, 줄간격 1.6
   - 항목 간격: 2.5mm
   - 강조 텍스트: Pretendard SemiBold, `#E8703A`
   - 서브 텍스트 (괄호): 8pt, `#9B8A7A`

   **문구**:
   1. 아래 QR 또는 **수성구 아이엠에스테틱**을 검색하여 신규 상담으로 예약해 주세요.
      *(예약금 3만 원이 있습니다)*
   2. 예약 후 보내드리는 앱으로 체질 및 피부 테스트를 진행해 주세요.
   3. 예약 시간에 맞추어 방문하시면 됩니다.
   4. 케어 후 리뷰를 작성해 주세요.

3. **케어 정보 박스**
   - 배경 `#FFF8F0`, 테두리 0.4pt `#F0E6D8`, 라운드 2.5mm
   - 패딩: 상하 3mm, 좌우 4mm
   - 라벨: Pretendard SemiBold, 9pt, `#9B8A7A`, 폭 16mm
   - 값: Pretendard SemiBold, 10.5pt, `#3D2E24`
   - 필드:
     - `받으실 케어` | `톤업 광채결케어`
     - `사용기한` | _______________ (밑줄 0.5pt `#D4A04A`, 최소 35mm)

4. **QR 섹션** (하단 영역, **가로 중앙 정렬**)
   - 상단 구분선: 0.4pt, `#E8DDD0`
   - QR 박스: 17mm × 17mm, 흰 배경, 테두리 0.3pt `#E8DDD0`, 라운드 1mm
   - QR 옆 텍스트:
     - "QR을 스캔하여 예약하세요": Pretendard SemiBold, 9pt, `#E8703A`
     - "네이버 > 수성구 아이엠에스테틱 / 신규 상담 예약": Pretendard Regular, 8pt, `#9B8A7A`
   - QR 박스와 텍스트 간격: 3mm

5. **푸터**
   - Pretendard Regular, 7.5pt, `#C4A882`, 자간 0.04em
   - 중앙 정렬
   - 문구: `대구 수성구 범어동 마크팰리스 2층 | @im.aesthetic.official`

## 작업 순서 추천

1. 캔버스 102×152mm 생성 (2페이지)
2. 브랜드 컬러 팔레트 등록 (6색)
3. 폰트 설정 (Pretendard / 본명조 / Playfair Display)
4. 앞면 → 배경 → 장식 → 로고 → 타이틀 → 본문 → 하단 순
5. 뒷면 → 헤더 → 4단계 안내 → 케어 박스 → QR → 푸터
6. QR 코드 생성 (미리캔버스 QR 요소 → 네이버 예약 URL)
7. **PDF/이미지 내보내기 (300dpi, CMYK 인쇄용)**
