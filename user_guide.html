1 로그인 (PIN 입력)

웹 브라우저에서 사이트에 접속하면 PIN 입력 화면이 나타납니다.

  1. 숫자 키패드에서 4자리 PIN을 입력합니다
  2. 올바른 PIN을 입력하면 자동으로 홈 화면으로 이동합니다
  3. PIN이 틀리면 빨간색으로 흔들림 효과가 나타납니다 — 다시 입력해주세요
Tip

PIN을 모르시면 담당자에게 문의해주세요. 보안을 위해 이 가이드에는 PIN을 기재하지 않습니다.

2 허브 메뉴 (홈 화면)

PIN 입력 후 나타나는 메인 화면입니다. 모든 페이지로 이동할 수 있는 중앙 허브 역할을 합니다.

홈 화면 — 전략 & 분석(상단) + 콘텐츠 도구(하단)

화면은 크게 두 영역으로 나뉩니다:

  1. Strategy & Analysis — 프로필 리뉴얼 가이드, 2026 트렌드 분석 등 전략 자료
  2. Tools & Pages — 카드뉴스, 하이라이트 커버, 이벤트 카드 등 제작 도구 10종

원하는 카드를 클릭하면 해당 페이지로 바로 이동합니다.

3 페이지 이동 (사이드바)

모든 페이지의 좌측 하단에 집 모양 버튼이 있습니다. 이 버튼을 누르면 사이드바 메뉴가 열립니다.

사이드바 메뉴 — 전체 페이지 목록 + 현재 페이지 하이라이트(주황색)
  1. 좌측 하단의 집 모양 버튼을 탭합니다
  2. 사이드바가 열리면 원하는 페이지를 선택합니다
  3. Home을 누르면 홈 화면으로 돌아갑니다
  4. 사이드바 바깥 영역을 탭하거나 ESC 키를 누르면 닫힙니다
Tip

현재 보고 있는 페이지는 주황색으로 표시되어 어디에 있는지 바로 알 수 있습니다.

4 전략 & 분석 자료

인스타그램 운영 방향을 확인하는 레퍼런스 문서입니다. 콘텐츠 제작 전에 먼저 확인하세요.

페이지내용
프로필 리뉴얼 가이드 인스타 프로필 비포&애프터 비교, 바이오 전략, 하이라이트 구성, 캡션 예시, 링크트리 전략
2026 트렌드 분석 경쟁 에스테틱 7개 계정 심층 분석, 트렌드 키워드, 전략 비교표, IM 전략 제언

이 자료는 읽기 전용입니다. 별도의 조작 없이 스크롤하며 확인하시면 됩니다.

5 콘텐츠 제작 도구

인스타그램에 올릴 이미지와 콘텐츠를 직접 만드는 도구들입니다.

페이지용도사용법
콘텐츠 대시보드 12주 콘텐츠 캘린더 주차별 포스트 확인, 수정, 상태 관리
피드 프리뷰 인스타 피드 미리보기 3열 그리드로 전체 피드 구성 확인
카드뉴스 메이커 카드뉴스 슬라이드 제작 캘린더에서 선택 또는 직접 입력 → 슬라이드 생성 → PNG 다운로드
하이라이트 커버 인스타 하이라이트 아이콘 6종 커버 중 원하는 것 다운로드 → 인스타 하이라이트 편집에서 적용
이벤트 카드 메이커 프로모션/가격표 카드 프리셋 선택 → 텍스트 수정 → PNG 다운로드
릴스 커버 메이커 릴스 커버 이미지 텍스트/배경 설정 → 1080x1920 PNG 다운로드
리뷰 오버레이 리뷰 피드 이미지 네이버/카톡 리뷰 텍스트 입력 → 오버레이 이미지 생성 → 다운로드
캔바 템플릿 Canva 연동 템플릿 10종 "Open in Canva" 클릭 → Canva에서 직접 편집
캐러셀 가이드 캐러셀 디자인 가이드 5가지 템플릿 디자인 방향 참고용
촬영 체크리스트 현장 촬영 가이드 체크박스 체크 + 메모 작성 (자동 저장됨)
카드뉴스 메이커 — 좌측에서 주차 선택 → 우측에 슬라이드 생성
하이라이트 커버 — 다운로드 버튼으로 개별/전체 다운로드

6 이미지 다운로드 & 활용

제작한 이미지를 인스타그램에 올리는 과정입니다.

  1. 각 메이커에서 이미지를 완성합니다
  2. 다운로드 또는 PNG 내보내기 버튼을 클릭합니다
  3. 다운로드된 이미지 파일을 확인합니다 (보통 Downloads 폴더)
  4. 인스타그램 앱에서 새 게시물을 만들고 이미지를 첨부합니다
  5. 대시보드에서 해당 포스트의 캡션과 해시태그를 복사하여 붙여넣습니다
Tip

카드뉴스 메이커에서는 "전체 다운로드" 버튼으로 모든 슬라이드를 한 번에 받을 수 있습니다. 개별 슬라이드 하단의 다운로드 아이콘으로 낱개 다운로드도 가능합니다.

7 유용한 팁

  1. 촬영 체크리스트는 체크 상태와 메모가 브라우저에 자동 저장됩니다. 현장에서 체크한 내용이 다음에 열어도 유지됩니다
  2. 캔바 템플릿은 "Open in Canva"를 눌러 Canva 사이트에서 직접 편집합니다. IM 브랜드 컬러가 이미 적용되어 있어 텍스트만 수정하면 됩니다
  3. 모든 이미지는 인스타그램 최적 사이즈로 생성됩니다 — 피드: 1080x1350 (4:5), 릴스/스토리: 1080x1920 (9:16)
  4. PC에서 작업 후 이미지를 카카오톡 나에게 보내기로 전송하면 모바일에서 바로 인스타에 올릴 수 있습니다
  5. 문제가 생기면 브라우저를 새로고침(F5)해보세요. 대부분의 표시 문제가 해결됩니다
