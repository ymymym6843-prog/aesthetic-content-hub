  • 미리캔버스 좌측 "텍스트" 패널 → 폰트 검색창에 "마루 부리" 또는 "Noto Sans"로 검색

  • 줄간격도 % 단위 (1.95 → 195%, 1.6 → 160%)

6. 작업 순서 추천

  1. 캔버스 102×152mm 생성 (2페이지)

  4. 앞면: 배경 → 모서리 장식 → 장미 → 로고 → 이름 → 타이틀 → 본문 → 하단
  5. 뒷면: 헤더 → 4단계 안내 → 케어 박스 → QR (중앙) → 푸터
  6. QR 코드 생성 (미리캔버스 QR 요소 → 네이버 예약 URL 입력)
  7. PDF 또는 이미지 내보내기 (300dpi, 인쇄용)

7. 체크리스트

  • ☐ 6색 팔레트 준수 (Orange-Gold 인접 금지)
