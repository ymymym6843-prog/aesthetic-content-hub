Canva Brand Kit Setup Guide

Canva Pro Brand Kit에 IMGROUND 브랜드를 등록하면 모든 템플릿에 일괄 적용할 수 있습니다.

Brand Kit 생성

Canva 홈 → 왼쪽 메뉴 Brand Kit → + 새 브랜드 → 이름: IMGROUND

Brand Color 등록 (11색)

아래 컬러를 순서대로 등록하세요:

#E8703A #FFF8F0 #F5EDE0 #D4A04A #3D2E24 #9B8A7A #D4837A #3A5A7C #7A9E7E

Brand Font 등록

제목: Playfair Display (Regular) / 부제목: Pretendard (SemiBold) / 본문: Pretendard (Regular)

템플릿 검색 → 커스터마이징

아래 카드의 "Canva에서 찾기" 버튼 클릭 → 마음에 드는 템플릿 선택 → Brand Kit 색상/폰트 적용 → IMGROUND 로고 추가

템플릿 링크 연동

완성된 템플릿의 공유 → 템플릿 링크 URL을 복사하여 이 페이지 관리자에게 전달하면 카드에 자동 연결됩니다.
