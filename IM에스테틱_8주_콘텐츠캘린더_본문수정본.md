모든 콘텐츠에 아래 3가지 키워드가 자연스럽게 녹아들도록 합니다:

### 7. 콘텐츠 디자인 전략

#### 핵심 원칙: "사람이 보이는 브랜딩"

2026년 인스타그램에서 효과적인 브랜딩은 **완벽하게 꾸민 이미지**가 아니라 **일관된 톤앤매너 + 실제 사람이 보이는 콘텐츠**입니다. 프로필 그리드를 봤을 때 "이 브랜드 톤이 확실하다"는 인상은 글래스모피즘이나 오버레이가 아닌, **색감·여백·폰트의 일관성**과 **전문가들의 존재감**으로 만들어집니다.

#### 단계별 전략 (브랜딩 → 신뢰 → 확장)

| 단계 | 주차 | 콘텐츠 비율 | 전략 |
|------|------|------------|------|
| **브랜드 소개** | 1~2W | 브랜딩 70% / 정보 30% | 팀·공간·시스템·전문가 소개로 "우리가 누군지" 각인. 릴스 없이 이미지/캐러셀만 |
| **신뢰 구축** | 3~5W | 브랜딩 30% / 정보 40% / 참여 30% | 첫 방문 안내, 후기·위생·데이터로 신뢰 증명. 3W부터 릴스 시작 |
| **전문성 + 확장** | 6~8W | 브랜딩 20% / 정보 40% / 참여 40% | 성분·미신타파 등 전문 정보로 저장/공유 유도. 웨딩·시즌 프로모션으로 확장 |

#### 1~2주차: 브랜드 소개기 (릴스 없음)
- **핵심 목표**: "IM 에스테틱 = 세 전문가의 따뜻한 통합 케어" 각인
- 1W: 팀 첫인사 + 공간 + 3인 케어 시스템 / 2W: 전문가 개별 소개
- 전문가 실제 사진 중심 (인물이 주인공, 꾸밈 최소화)
- 색감·여백·폰트만으로 통일감 (오버레이 효과 지양)
- 첫 슬라이드 = 실제 사진 + 간결한 타이포 (1~2줄)
- CTA는 본문 카피에서만 자연스럽게
- **그리드 전략**: Row 1(1W) = 단체/공간/시스템, Row 2(2W) = 유수정/우연우/김은경

#### 3~5주차: 신뢰 구축기 (3W부터 릴스 시작)
- 3W: 첫 방문 안내 + 릴스 시작 (전환기)
- 후기·위생·시술 데이터 등 **증거 기반** 콘텐츠
- 카드뉴스(인포그래픽) 본격 도입 — 저장 유도
- 후킹 유형: `공감 질문형` / `대화 인용` (자극적 클릭베이트 금지)
- 참여형(체크리스트, 자가진단) 콘텐츠로 댓글·DM 유도

#### 6~8주차: 전문성 + 확장기
- 1슬라이드 = **8~10단어 이내의 한 줄 후킹**으로 시선 고정
- 후킹 유형: `의외의 팩트` / `공감 질문형` / `대화 인용`
- 마지막 슬라이드 = 저장/공유/DM CTA 전용 슬라이드
- 웨딩·시즌 등 타겟 확장 콘텐츠

#### IM 에스테틱 디자인 스타일
- **컬러**: 웜 화이트(#FFF8F0) 배경 + 차콜(#333) 텍스트 + 오렌지(#FF8C42) 포인트 (5% 이내)
- **폰트**: 제목 36px+ Bold / 본문 24px+ Regular (Pretendard 또는 Noto Sans KR)
- **레이아웃**: 슬라이드당 핵심 1개, 10~15단어 이내, 충분한 여백 (70%+)
- **사진 톤**: 따뜻한 자연광, 약간의 웜톤 보정, 과도한 필터 금지
- **디자인 포맷**:
  - `리얼 스냅형` — 실제 촬영 사진 + 최소 타이포 (1~2줄). 인물·공간·시술 장면 (**전 주차 공통**)
  - `카드 인포그래픽형` — 웜화이트 배경 + 정보 카드. 체크리스트, 비교표, 프로세스 등 (**3W부터 도입**)
  - `질문→답변 플립형` — 1장 질문(다크), 2장 답변(밝은 배경) 반복 (**6W 이후**)
  - `후기/UGC형` — 고객 후기 캡처 + 브랜드 프레임. 실제 카톡/네이버 후기 활용 (**4W부터**)

### 8. 캐러셀 슬라이드 설계 원칙 (목요일 카드뉴스 적용)

#### 기본 규칙
- **슬라이드 수**: 모든 목요일 카드뉴스는 최소 **5장, 최대 7장**으로 구성
- **첫 장 = 후킹 전용**: 8~10단어 이내의 강렬한 한 줄. 탐색탭에서 스크롤을 멈추게 하는 역할
- **마지막 장 = CTA 전용**: 저장/공유/예약 유도 카드. 로고 + 연락처 + 북마크 아이콘
- **슬라이드당 핵심 1개**: 한 장에 정보 2개 이상 넣지 않기 (3초 안에 읽을 수 있어야 함)
- **텍스트**: 슬라이드당 10~15단어 이내, 여백 70%+ 유지
- **페이지 인디케이터**: 모든 슬라이드 우상단에 "1/6" 형식 번호 표기 → 완독률 ↑

| 템플릿 | 슬라이드 수 | 적용 콘텐츠 | 구조 |

| **카드 인포그래픽형** | 6장 | 팁, 체크리스트, 성분, 과정, 프로모션 | 후킹→공감→솔루션1→솔루션2→솔루션3→CTA |
